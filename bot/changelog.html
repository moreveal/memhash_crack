Список изменений Все основные изменения плагина будут документироваться здесь. [1.1.0] - 2024-12-17 
Улучшено:
- Система лицензий (один билд - несколько ключей)
[1.0.9] - 2024-12-15 
Изменено:
- Концепция оплаты времени, чтобы позволить распределять подписку
между разными аккаунтами

Удалено:
- Возможность приобретения бесконечной подписки
[1.0.8] - 2024-12-12 
Добавлено:
- Возможность установки используемой мощности процессора
- Реферальная система
- Список последних изменений (/changelog)

Улучшено:
- Удобный инструмент для автоматизированного исправления плагина под
новые версии приложения

Изменено:
- Минимальный уровень энергии для отправки валидных блоков повышен
- Корректировка цен в соответствии с ожидаемым уровнем доходности
[1.0.7] - 2024-12-11 
Исправлено:
- Поддержка TelegramID пользователя как x64 числа
- Прочие незначительные улучшения
[1.0.6] - 2024-12-10 
Изменено:
- Корректировка хеширования в соответствии с последним обновлением MemHash
(Nitrocore)

Исправлено:
- Размер приветственного сообщения уменьшен
[1.0.5] - 2024-12-09 
Добавлено:
- Поддержка уникальных предложений: индивидуальное назначение скидок

Изменено:
- Дополнительная информация в приветственном сообщении
- Скрытие назойливой ошибки в консоли (появилась после обновления MemHash)

Исправлено:
- Энергия теперь не уменьшается вовсе
- Исправление нескольких опечаток
[1.0.4] - 2024-12-08 
Добавлено:
- Воркер переписан на C++ (x50000 разница в скорости по сравнению с WebASM)

Изменено:
- Замеры скоростей в приветственном сообщении
[1.0.3] - 2024-12-06 
Исправлено:
- Шанс отправки валидных блоков
[1.0.2] - 2024-12-05 
Добавлено:
- Возможность настройки шанса отправки валидных блоков
- Дополнительная небольшая оптимизация алгоритма

Изменено:
- Удаление лимита на 200 shares за блок (добавлен в последнем обновлении MemHash)

Исправлено:
- Обновление в соответствии с новой версией приложения
- Правильное распределение воркеров (без генерации одинаковых nonce)
- Исправление алгоритма хеширования (иногда генерировал неверные хеши)
[1.0.1] - 2024-12-04 
Изменено:
- Дополнительная информация в приветственном сообщении

Исправлено:
- Определения Telegram ID при создании билда
- Выдача тестовой версии билда
[1.0.0] - 2024-12-02 
Добавлено:
- Создание бота
- Оптимизированный алгоритм вычисления SHA-256 (x8 от стандартной скорости)
